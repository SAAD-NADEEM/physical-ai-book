# Docusaurus-based Technical Book on Physical AI and Humanoid Robotics

## Project Overview
- Build a 40,000-60,000 word technical book for a 13-week graduate course
- 4 modules covering ROS 2, Gazebo/Unity, NVIDIA Isaac, and Vision-Language-Action
- Deploy as static site on GitHub Pages
- Full compliance with 30 constitutional principles

## Module Structure
### Module 1: ROS 2 Fundamentals (Weeks 1-5)
- Chapter 1: Introduction to ROS 2 and Robot Architecture
- Chapter 2: Nodes, Topics, Services, and Actions
- Chapter 3: Robot Manipulation and Control
- Chapter 4: Navigation and Path Planning
- Chapter 5: Multi-Robot Systems and Coordination

### Module 2: Simulation Environments (Weeks 6-7)
- Chapter 6: Physics Simulation with Gazebo
- Chapter 7: Unity Integration and Visual Simulation

### Module 3: NVIDIA Isaac Platform (Weeks 8-10)
- Chapter 8: Introduction to Isaac Sim and Isaac ROS
- Chapter 9: Perception and Sensing with Isaac
- Chapter 10: Navigation and Manipulation Frameworks

### Module 4: Vision-Language-Action Integration (Weeks 11-13)
- Chapter 11: Vision-Language Models for Robotics
- Chapter 12: Action Generation and Execution
- Chapter 13: Capstone Project Integrating All Modules
